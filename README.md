# 💰 Personal Expense Tracker

## 📌 Project Description
A simple Personal Expense Tracker built using Python and Streamlit. It helps users record daily expenses, categorize them, and view saved expense records.

## ✨ Features
- Add new expenses
- Enter expense amount
- Select category
- Automatically save the current date
- View all saved expenses
- Simple and user-friendly Streamlit interface

## 🛠️ Technologies Used
- Python
- Streamlit
- File Handling

## 🚀 Future Improvements
- Expense summary
- Charts and graphs
- Monthly reports
- Search and filter
- Delete/Edit expenses
